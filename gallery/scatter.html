Basic scatterplot

One mark per observation — car weight against fuel economy, the mtcars dataset. Left is the R + ggplot2 original; right is the same figure from cinderplot: no R session, one command, vector SVG out.

ggplot2 · R reference
cinderplot · C svg

R — the ggplot2 way

# needs an R session
library(ggplot2)
ggplot(mtcars,
       aes(x = wt, y = mpg)) +
  geom_point()

$ — the cinderplot way

# one C binary, no R
cinderplot data/mtcars.csv \
  -x wt -y mpg -o scatter.svg

# …or the grammar form
cinderplot 'data/mtcars.csv
  + aes(wt, mpg) + geom_point()'
Same figure, no R. cinderplot reproduces ggplot2's theme_gray faithfully — panel fill, gridlines, axis breaks, and point geometry — from a single C binary that reads the CSV and writes PDF or SVG in milliseconds.
