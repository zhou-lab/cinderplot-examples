Basic scatterplot

The most basic scatterplot: one mark per observation, weight against fuel economy from the mtcars dataset. On the left, the R + ggplot2 original; on the right, the same figure drawn by cinderplot — no R session, one command, vector SVG.

ggplot2 · R
cinderplot SVG

The ggplot2 way

# R
library(ggplot2)
ggplot(mtcars, aes(x = wt, y = mpg)) + geom_point()

The cinderplot way

# shell — no R needed
cinderplot data/mtcars.csv -x wt -y mpg -o scatter.svg

# or the grammar form
cinderplot 'data/mtcars.csv + aes(wt, mpg) + geom_point()' -o scatter.svg
Same figure, no R. cinderplot reproduces ggplot2's theme_gray faithfully — panel fill, gridlines, axis breaks, and point geometry — from a single C binary that reads the CSV and writes a PDF or SVG in milliseconds.
Data: data/mtcars.csv · reference rendered with gallery/scatter/gen.R. Part of cinderplot.
