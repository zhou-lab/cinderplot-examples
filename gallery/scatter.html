Basic scatterplot

One mark per observation — car weight against fuel economy, the mtcars dataset. Hover the plot to see the code. Toggle how it's drawn — the cinderplot command or the R original — it's the same figure either way.

hover for code ↓ 
# one C binary, no R session
cinderplot data/mtcars.csv \
  -x wt -y mpg -o scatter.svg

# …or the grammar form
cinderplot 'data/mtcars.csv
  + aes(wt, mpg) + geom_point()'
# needs an R session
library(ggplot2)
ggplot(mtcars,
  aes(x = wt, y = mpg)) +
  geom_point()
Same figure, no R. cinderplot reproduces ggplot2's theme_gray faithfully — panel fill, gridlines, axis breaks, and point geometry — from a single C binary that reads the CSV and writes PDF or SVG in milliseconds.
